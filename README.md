# Yash Agrawal - Portfolio

Personal portfolio showcasing my work as an AI/ML Engineer.

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Vite

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Customization

All portfolio content is centralized in `src/data/portfolio.ts`. Edit this single file to update:

- Personal info & social links
- Work experience
- Projects
- Skills
- Education
- Blog configuration
